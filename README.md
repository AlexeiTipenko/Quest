## Getting Started

This game was developed on Unity 2017.3.0 on the MacOS Environment

### Prerequisites

To get started, you need to install Unity, head to

```
- Installation
https://unity3d.com/get-unity/download

- Cloning
git clone https://github.com/JKirkYuan/Quest.git
```
